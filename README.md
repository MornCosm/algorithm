1. [两数之和](https://leetcode-cn.com/problems/two-sum/)<br>
题目难度:
```text
简单
```
题目描述:
```text
    给定一个整数数组 nums 和一个目标值 target ， 请你从该数组中找出和为目标值的那两个整数，并返回他们的数组下标。
    你可以假设每种输入只对应一个答案，但是，你不能重复利用这个数组中同样的元素    
```
示例:
```text
给定 nums = [2, 7, 11, 15], target = 9
因为 nums【0】 + nums[1] = 2 + 7 =9
所以返回 [0, 1]

```
